Tagging and Assigning Report Attributes

1. Click "Reports"

2. Click on the report you'd like to update

3. Click "Details"

How to update the tags

4. Click the drop down next to the Tags title

5. Click the tag you'd like to add to the report.

How to add a custom tag

Alert: Alert! This will only work for some attribute boxes. To test this, type in the box and if you get an error, it is not available.

6. Click in the box under the tag / attribute you'd like to update.

7. Type the title of your tag you'd like to add, and hit enter.

How to update report attributes

8. Click the on the drop down of the report attribute you'd like to update

9. Click on the value you'd like to assign from the drop down

Tip: Tip! You can usually assign more than one value. Follow the steps below.

10. Click on the same report attribute drop down as before

11. Click on the other value you'd like to assign to the report.

12. Repeat the steps above as many times as needed.
